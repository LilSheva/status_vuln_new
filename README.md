# Анализатор Статусов Уязвимостей v2.0

Два десктопных приложения для автоматизации анализа уязвимостей: **Сопоставитель** (анализ ТСУ vs ППТС) и **База Знаний** (менеджер правил). Связь между ними — через общий SQLite-файл.

## Быстрый старт

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск

```bash
# Сопоставитель
python -m matcher.main

# База Знаний
python -m knowledge_base.main
```

### Тесты

```bash
pytest tests/ -v
```

## Архитектура

```
[Сопоставитель]  ←→  [knowledge.db (SQLite)]  ←→  [База Знаний]
```

- **Сопоставитель** — загружает ТСУ и ППТС файлы (Excel/CSV), прогоняет через пайплайн (векторный поиск + fuzzy + exact), генерирует XLSX-отчёт
- **База Знаний** — управление правилами сопоставления (CRUD, поиск, фильтрация, тестирование)
- Приложения общаются только через SQLite — никаких прямых связей

## Пайплайн Сопоставителя

```
Загрузка ТСУ + ППТС
        ↓
Препроцессинг (плагины из scripts/)
        ↓
Проверка по Базе Знаний (exact → contains → regex → vector)
        ↓
Векторный поиск (sentence-transformers + cosine similarity)
        ↓
Нормализация (транслитерация кириллица ↔ латиница)
        ↓
Fuzzy-сравнение (RapidFuzz)
        ↓
Прямое сопоставление (exact/substring)
        ↓
Комбинированный скоринг (50% vector + 30% fuzzy + 20% exact)
        ↓
Присвоение статусов → XLSX-отчёт (3 листа)
```

## Статусы

| Статус | Описание | Источник |
|--------|----------|----------|
| ДА | ПО есть в инфраструктуре | Только через базу знаний |
| НЕТ | ПО нет в инфраструктуре | Автоматически / БЗ |
| ЛИНУКС | Linux-специфичное ПО | БЗ |
| УСЛОВНО | Требует доп. проверки | БЗ |
| (пусто) | Есть кандидаты, уверенности недостаточно | Ручной разбор |

## Плагины препроцессинга

Папка `scripts/` содержит плагины для обработки данных перед анализом:

| Плагин | Описание |
|--------|----------|
| `split_multiproduct.py` | Разбивка мультипродуктовых строк (разделители: `;`, ` и `, `/`) |
| `clean_microsoft.py` | Очистка Microsoft-специфичного мусора ((R), (TM), editions, KB...) |
| `clean_versions.py` | Удаление номеров версий и годов из названий продуктов |
| `_template.py` | Шаблон для создания новых плагинов |

## Сборка в EXE

```bash
pyinstaller matcher.spec
pyinstaller knowledge_base.spec
```

Результат: `dist/Сопоставитель/` и `dist/База Знаний/`.

## Стек

- Python 3.11+
- PySide6 (GUI)
- sentence-transformers (векторный поиск)
- scikit-learn (cosine similarity)
- RapidFuzz (нечёткое сравнение строк)
- transliterate (транслитерация)
- openpyxl (XLSX-отчёты)
- SQLite3 (база знаний)
